Timestamp Microservice

How to Use

Can pass either a date in Month Day Year format or as a unix timestamp and both of them back in a json object. Can use full month name or abbreviated name.


Be sure to leave space or commas between the date values.

A few example calls:
  • https://timestampcallback.glitch.me/June 21, 1999
  • https://timestampcallback.glitch.me/Feb%2017%202004
  • https://timestampcallback.glitch.me/1234823872
The responses ready to go in json format:
  • {"unix": 929923200, "natural": "june 21, 1999"}
  • {"unix": 1076976000, "natural": "february 17, 2004"}
  • {"unix": 1234823872, "natural": "february 16, 2009"}

Project made by Ben Wakefield for freecodecamp's timestamp microservice backend project. More info on Github.
